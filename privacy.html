天气岛 隐私政策

生效日期:2026-05-13 · 版本:1.0 · App Bundle ID:com.wisstiger.weatherislandapp

欢迎使用《天气岛》(以下简称"本应用")。我们(Beijing Shengtenghui Technology Co., Ltd.,以下简称"我们")极度重视您的隐私。本政策说明本应用如何收集、使用和保护您的信息。

1. 我们收集的信息

本应用仅收集与天气功能相关的最少信息:

  • 地理位置(可选):当您允许定位权限时,本应用会获取您当前的地理位置(经度、纬度),用于向天气服务请求您所在城市的实时天气数据。位置信息仅本地使用,不会上传到我们的服务器。
  • 城市偏好:您手动选择的城市保留在本机存储中,用于跨启动保留设置。
  • 购买状态:您的 In-App Purchase(应用内购买)凭证由 Apple 系统管理。我们仅通过 Apple StoreKit 验证您的购买状态以解锁完整功能。

本应用不收集以下信息:

  • 姓名、手机号、邮箱、身份证号等可识别个人身份的信息
  • 设备 IDFA / IDFV / 设备指纹
  • 通讯录、相册、麦克风、相机内容
  • 账号注册或用户行为追踪数据

2. 信息如何使用

  • 地理位置 → 调用第三方天气 API(和风天气 QWeather)获取实时天气;请求中包含经纬度,响应在本地呈现。
  • 城市偏好 → 本机存储,跨启动恢复。
  • 购买凭证 → Apple StoreKit 校验,解锁完整功能。

3. 信息共享

本应用不与任何第三方分享您的个人信息。仅向以下服务发送必要的、不可识别您个人身份的数据:

  • 和风天气 QWeather(隐私政策):接收您的经纬度并返回天气数据。请求中不含您的姓名、设备 ID 等个人信息。
  • Apple iCloud / StoreKit:Apple 系统服务,管理您的 IAP 购买凭证。

4. 数据存储与安全

  • 本应用使用 iOS 系统的 App Group / UserDefaults 保留城市偏好与购买状态,数据存储在您的设备本地。
  • 我们不在自有服务器上存储您的任何数据。
  • 所有第三方 API 通信均使用 HTTPS 加密。

5. 子组件 / 小组件 (Widget)

本应用在 iOS 主屏与锁屏的 Widget 显示当前天气信息,数据来源与主应用一致(本地缓存 + 和风天气 API),不收集额外信息。

6. 儿童隐私

本应用不针对 13 岁以下儿童设计。如果您是 13 岁以下儿童的监护人,发现儿童在未经允许的情况下使用本应用,请联系我们删除相关本地数据。

7. 您的权利

  • 撤回定位授权:在 iOS 设置 → 隐私与安全性 → 定位服务 → 天气岛 → 选择"永不",本应用将自动 fallback 显示北京天气。
  • 清除本地数据:卸载本应用,所有本地数据(城市偏好、购买凭证缓存)随之删除。Apple iCloud 中的购买记录不删除(您可通过同 Apple ID 重新安装恢复购买)。

8. 政策变更

本政策可能随产品功能演进而更新,任何更新会同步发布在本页面。建议您不定期回访本页面。
